Class: IndexedSequence

IndexedSequence

new IndexedSequence()

An IndexedSequence is a Sequence that provides random access to its elements. This extends the API for iterating with the additional methods get and {@link length}, allowing a sequence to act as a "view" into a collection or other indexed data source.
